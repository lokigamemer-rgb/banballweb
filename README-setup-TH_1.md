# คู่มือตั้งค่าเว็บสตูดิโอเสียงพากย์

ไฟล์เว็บคือ `index.html` ไฟล์เดียวจบ เปิดในเบราว์เซอร์ได้เลย หรืออัปโหลดขึ้นโฮสต์ฟรีอย่าง Netlify / Vercel / GitHub Pages / Cloudflare Pages

ทุกอย่างที่ต้องแก้รวมอยู่ในกล่อง `CONFIG` ใกล้ ๆ ท้ายไฟล์ (มองหาคำว่า `const CONFIG`)

---

## 1) แก้ข้อมูลร้าน + ช่องทางติดต่อ

```js
const CONFIG = {
  studioName: "ชื่อสตูดิโอของคุณ",
  email:    "hello@yourstudio.com",
  lineId:   "@yourline",        // LINE OA id
  wechatId: "your_wechat_id",   // 微信号
  whatsapp: "+66800000000",     // ใส่รหัสประเทศด้วย
  ...
```

## 2) เชื่อม Google Drive (อัปเดตเสียงเองโดยไม่ต้องแก้โค้ด)

หลักการ: ทำโฟลเดอร์ใน Drive ไว้ 1 โฟลเดอร์ → โยนไฟล์เสียงเข้าไป → เว็บดึงมาแสดงอัตโนมัติ
อยากเพิ่ม/เปลี่ยนเสียงทีหลัง แค่ลากไฟล์เข้าโฟลเดอร์ ไม่ต้องแตะโค้ดอีกเลย

**ขั้นตอน**

1. สร้างโฟลเดอร์ใน Google Drive แล้วเอาไฟล์เสียง (`.mp3` / `.wav`) ใส่เข้าไป
2. คลิกขวาที่โฟลเดอร์ → **แชร์** → ตั้งเป็น **"ทุกคนที่มีลิงก์ (Anyone with the link) — ผู้อ่าน (Viewer)"**
3. หา **Folder ID**: เปิดโฟลเดอร์ในเบราว์เซอร์ ดูที่ URL
   `https://drive.google.com/drive/folders/`**`1AbCdEfGhIJklmNoP`** ← ส่วนหลัง `/folders/` คือ ID
4. สร้าง **API Key**:
   - เข้า https://console.cloud.google.com → สร้าง Project (หรือใช้อันเดิม)
   - เมนู **APIs & Services → Library** → ค้น **Google Drive API** → กด **Enable**
   - ไป **APIs & Services → Credentials → Create Credentials → API key** แล้วก๊อปคีย์มา
   - (แนะนำ) กด **Edit API key → Application restrictions** จำกัดเฉพาะโดเมนเว็บคุณ และ **API restrictions** เลือกเฉพาะ Google Drive API เพื่อความปลอดภัย
5. เอาทั้งสองค่ามาใส่ใน CONFIG:

```js
  driveApiKey:   "AIzaSy.....",
  driveFolderId: "1AbCdEfGhIJklmNoP",
```

> ถ้าเว้น 2 ค่านี้ว่างไว้ เว็บจะแสดง "เสียงเดโม" ที่ฝังมาให้แทน ใช้พรีวิวหน้าตาได้ทันที

## 3) วิธีตั้งชื่อไฟล์เสียง (เพื่อให้ขึ้นป้ายภาษา/ชื่อ/สไตล์)

ใช้ขีดล่างคู่ `__` คั่น 4 ส่วน:

```
ภาษา__ชื่อนักพากย์__สไตล์__ชื่อชิ้นงาน.mp3
```

ตัวอย่าง:

| ตั้งชื่อไฟล์แบบนี้ | จะแสดงเป็น |
|---|---|
| `EN__Daniel__Character__Anime Hero.mp3` | ป้าย EN · Daniel · ตัวละคร |
| `ZH__美玲__Drama__竖屏短剧 女主.mp3` | ป้าย 中文 · 美玲 · ดราม่า |
| `TH__ฟ้า__Friendly__พากย์การ์ตูน.mp3` | ป้าย ไทย · ฟ้า · เป็นกันเอง |

- **ภาษา** รองรับ: `TH` `EN` `ZH`(หรือ `CN`) `JP`
- **สไตล์** ที่แปลอัตโนมัติ: `Character` `Drama` `Narration` `Energetic` `Warm` `Corporate` `Luxury` `Friendly` `Trailer`
  (พิมพ์คำอื่นก็ได้ แค่จะไม่ถูกแปลตามภาษา UI)
- ถ้าไม่อยากตั้งตามรูปแบบนี้ ใส่ชื่อธรรมดาก็ได้ ระบบจะเอาชื่อไฟล์มาแสดงตรง ๆ

## 4) เพิ่ม / แก้ผลงานของสตูดิโอ

ในกล่องสคริปต์มองหา `const PORTFOLIO` แล้วเพิ่มหรือลบรายการได้เลย แต่ละชิ้นมี 5 ช่อง:

```js
const PORTFOLIO = [
  { title:"ชื่อผลงาน", cat:"หมวด · บทบาท", year:"2025", image:"", link:"" },
];
```

- **title** — ชื่อผลงาน
- **cat** — หมวด/บทบาท เช่น `Anime · Thai dub` (พิมพ์อิสระ)
- **year** — ปี
- **image** — ลิงก์รูปปก (ใส่ก็ได้/เว้นว่างได้) ถ้าเว้นว่างจะขึ้นไอคอนฟิล์มแทน
  ใช้ลิงก์รูปจาก Google Drive ได้ ด้วยรูปแบบ `https://drive.google.com/thumbnail?id=FILE_ID&sz=w800`
- **link** — ลิงก์ดูผลงานเต็ม (YouTube/Bilibili ฯลฯ) ถ้าใส่ การ์ดจะกดเข้าได้และมีปุ่มเล่น

เว้น `image` กับ `link` ว่างไว้ก็ได้ การ์ดยังแสดงสวยงาม

## 5) ระบบเปลี่ยนภาษา

ปุ่ม ไทย / EN / 中文 มุมขวาบน เว็บจะ **เดาภาษาจากเบราว์เซอร์ผู้เข้าชมให้อัตโนมัติ**
(จีน→中文, ไทย→ไทย, อื่น ๆ→English) และจำผ่านลิงก์ได้ เช่น `เว็บคุณ.com/#zh`

---

## หมายเหตุเรื่องไฟล์เสียงจาก Drive
ไฟล์เสียงควรมีขนาดพอเหมาะ (เช่น MP3 คุณภาพดีตัวอย่างสั้น ๆ) จะสตรีมได้ลื่น
ถ้าไฟล์ใหญ่มาก Google อาจสตรีมช้า แนะนำทำเป็นไฟล์ตัวอย่าง 15–40 วินาทีต่อชิ้น
